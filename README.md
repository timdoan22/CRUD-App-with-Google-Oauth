# Project Title

Stories

## Description

A simple full stack project developed to implement CRUD operations and logging in using Google Oauth via Passport.JS.<br />  
Built with HTML, CSS, EJS, Javascript, Node.JS, Express.JS, MongoDB and Passport.JS
